Introduction To CSS

What is CSS

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML.

How To Add CSS

There are three ways to add CSS:

  1. Inline CSS
  2. Internal CSS
  3. External CSS

Inline CSS

An inline style may be used to apply a unique style for a single element. It is are defined within the "style" attribute of the relevant element. Inlining styles has been added to this webpage.
